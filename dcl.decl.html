Insert the following paragraph after /1

A declarator that declares generic function () may include a requires-clause. A program that includes a requires-clause for any other declaration is ill-formed. Let C be a variable concept constraining a single type parameter: void f1(auto x) requires C<decltyp(x)>; // Ok void f2(int x) requires C<int>; // Error: f2 is not a generic function auto n requires C<decltype(n)> = g(); // Error: cannot constrain variable declarattion struct S { } requires C<S>; // Error: cannot constrain a class declaration

The requires-clause associates its constraint with its template declaration ().

Meaning of declarators

Functions

Add the following paragraphs after /14.

A generic function is a function template whose template-parameter-list has a parameter-declaration whose type-specifier is either auto or a constrained-type-name. auto f(auto x); // Ok void sort(C& c); // Ok (assuming C names a concept)

The declaration of a generic function has a template-parameter-list that consists of one invented type template-parameter for each occurrence of auto or each unique occurence of a constrained-type-name in the function's parameter-declaration-clause, in order of appearance. The invented type of template-parameter is a parameter pack if the corresponding parameter-declaration declares a function parameter pack (). The associated constraints of a generic function are the conjunction of constraints introduced by the use of constrained-type-names in the parameter-declaration-clause and the requires-clause in the function's declarator, if present. The following generic function declarations are equivalent: auto f(auto x, const C& y); template<typename T1, typename T2> requires C<T2>() auto f(T1 x, const T2& y);

All placeholder types introduced using the same constrained-type-name have the same invented template parameter. The following generic function declarations are equivalent: auto g(C a, C* b); template<C T> auto g(T a, T* b);

If an entity is declared by an abbreviated template declaration, then all its declarations must have the same form.
