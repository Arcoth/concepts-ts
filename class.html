A requires-clause shall only appear in a member-declarator if its declarator is a function declarator. The requires-clause associates its constraint-expression with the function. template<typename T> concept bool C = true; struct A { A(int*) requires C<int>; // OK void f() requires C<int>; // OK int x requires C<T> // error: constrained member viarable };
